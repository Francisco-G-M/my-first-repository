¿Qué es la programación de algoritmos básicos?

La programación de algoritmos básicos se basan en los fundamentos del pensamiento algorítmico,.A su vez estos algoritmos básicos son un conjunto ordenado y determinado de operaciones que permiten la solución de un problema.

Convertir Celsius a Fahrenheit

Para convertir Celsius a Fahrenheit usamos la siguiente formula: Celsius múltiplicado por 9/5 + 32:

    
    function convertirTemperaturas(celsius) {
    const FAHRENHEIT = celsius*9/5 + 32;
    return FAHRENHEIT;
    }
    let final = convertirTemperaturas(30);
    console.log(final);

Invertir una cadena

En caso de que quisieramos invertir una cadena podemos hacerlo mediante un ciclo for:

    
    function invertir(string) {
    let newArr = "";
    for(let i = string.length - 1; i >= 0; i--) {
    newArr += string[i]; 
    }
    return newArr;
    }
    let final = invertir("hola");
    console.log(final);

Factorizar un número

Recordemos que un factorial es el producto de todos los enteros menores o iguales al número seleccionado,en este caso vamos a devolver el factorial de 9:

    
    function factorizar(n) {
    if(n <= 0) {
    return 1;

    console.log(n);

    console.log(factirizar(9));

Encontrar la palabra más larga de una cadena

En caso de que quisieramos encontrar la palabra más larga de una cadena haríamos así:

    
    function theLongWord(str) {
    let longWord = "";
    let word = str.split(" ");
    for(let c = 0; c < word.length; c++) {
    let individualWord = word[c];
    if(individualWord.length > longWord.length) {
    longWord = individualWord;
    }
    }
    return longWord.length;
    }
    let buscar = theLongWord("Cual es la palabra más larga");
    console.log(buscar);
    /*Lo que hacemos es iterar a través de la cadena hasta encontrar la palabra más larga,".split()" lo que hace es separar cada palabra
    en una subcadena dentroo de un arreglo,si ponemos (" ") con un espacio hará lo ya dicho, si no hará lo mismo pero separando y volviendo
    una subcadena a cada letra*/

Devolver los números mayores de los arreglos

Para devolver un arreglo que tenga el número mayor de cada matriz anidad,podemos iterar con un bucle for: 

    
    function ejemplo(array) {
    let newArr = [];
    for(let h = 0; h < array.length; h++) {
    let largestNumber = array[h][0];
    for(let i = 0; i < array[h].length; i++) {
    if(array[h][i] > largestNumber) {
    largestNumber = array[h][i];
    }
    }
    newArr[h] = largestNumber;
    }
    return  newArr;
    }
    var final = ejemplo([[8, 6, 9, 7], [13, 30, 29, 21], [20, 15, 14, 10], [0, 99, 8, 3]]);
    console.log(final);

Para verificar que una cadena termine con la cadena final dada podemos hacer lo siguiente:

    
    function ending(str, end) {
    let theEnd = end.length;
    let endMatch = str.splice(str.length - theEnd, str.length);

    return false;

    var theFinal = ending("Tomas", "s");

Repetir una cadena

Para repetir una cadena por una determinada cantidad de veces podemos hacerlo de las siguientes maneras: 

    
    function repeatString(str1, num) {
    let string = "";
    for(let i = 0; i < num; i++) {
    string += str1;
    }
    return string;
    }
    repeatString("W", 3);
    function example2(str2, n) {
    return n > 0 ? str - example2(str2, n - 1) : "";
    }
    console.log(example2(363, 2));
    //La forma más sencilla y práctica para esto sería usando el método .repeat():
    function example3(str3, n2) {
    return str3.repeat(n2);
    }
    console.log(example3("hi", 4));

Recortar una cadena

En caso de que quisieramos recortar una cadena que es más larga de la longitud máxima asignada,podemos usar .splice() para truncarla donde querramos:

    
    function truncar(str, n) {
    return string.length > n ? string.splice(0, n) : string;
    }
    console.log(truncar("Truncar la palabra más larga", 7)); //Esto nos devuelve la primera palabra

Convertir la primera letra en Mayúscula

En esta ocasión queremos convertir en Mayúscula las primeras letras de las palabras de una cadena:

    
    function letrasConMayuscula(str) {
    let newString = str.split(" ");
    let newArr = [];
    for(let x in newString) {
    newArr[x] = newString[x][0].toUpperCase() + newString[x].slice(1).toLowerCase();
    }
    return newArr.join(" ");
    }
    let a = letrasConMayuscula("soy una tasa de cafe");
    console.log(a);
    /* .toLowerCase() devuelve una copia de la cadena en minuscula,en este caso,vuelve a minuscula todo el contenido de la subcadena a partir
    del índice 1 de esa subcadena.
    .join(" ") concatena los elementos de una o más matrices unidimensionales,en este caso esta uniendo las subcadenas hechas por .split(" ")
    .Este método toma un parametro: ejemplo.join(separador); el separador es opcional,si se omite,los elementos del arreglo seran separados
    por una coma, pero si es una cadena vacía ("") une todos los elementos sin ningún carácter entre ellos*/

Copiar y agregar un arreglo en otro arreglo

En este caso queremos copiar y agregar un arreglo en otro arreglo,insertando en el índice índicado como parametro en la función:

    
    function copiarAgregar(arr1, arr2, indice) {
    let newArr = arr2.slice();
    for(let w = 0; w < arr1.length; w++) {
    newArr.splice(indice, 0, arr1[w]);
    indice++;
    }
    return newArr;
    }
    let a = copiarAgregar([55, 66, 77], [44, 88, 99], 1);
    console.log(a);
    //El parametro "índice" índica donde va a empezar a copiar el arr1 en el arr2

Eliminar los valores falsos de un arreglo con .filter()

El método .filter() tiene como función crear un nuevo Array con los elementos que cumplen con la condición dada (.filter(consición)),en este caso,la condición son los valores booleanos primitivos (true, false),devolviendonos solo los elementos que sean true:

    
    function convertirValoresEnBooleanos(array) {
    return array.filter(boolean);
    }
    let a = convertirValoresEnBooleanos([10, "Hi", '', NaN, 9, false]);
    console.log(a);
    //Aquí nos devuelve loa elementos considerados true: [10, 'Hi', 9],todos los demas son false

Ordenar y manipular los elementos de un array con el método .sort()

La función de .sort() es ordenar los elementos de un array según un criterio(este mismo puede ser definido opcionalmente con un callback,estos son básicamente el uso de funciones como párametros de otras funciones).En esta ocasión,queremos devolver el índice más bajo en el que un valor va a ser introducido en un arreglo que anteriormente fue ordenado:

    
    function example(array, n) {
    return array.concat(n).sort((x, y) => x-y).indexOf(n);
    }
    let result = example([5, 7], 6);
    console.log(result);
    //Esta llamada nos devuelve el índice más bajo donde fue insertado el valor de n en el array

Comprobar la existencia de los elementos del segundo índice de un arreglo en el primer índice

En esta ocasión vamos a comprobar que los elementos del segundo índice se encuentren dentro del primer índice de un arreglo:

    
    function comprobar(array) {
    let first = array[0].toLowerCase();
    let second = array[1].toLowerCase();
    for(let a = 0; a < second.length; a++) {
    if(first.indexOf(second[a]) < 0) {
    return false;
    }
    }
    return true;
    }
    comprobar(["buenas", "HOLA"]);

Dividir un arreglo unidimensional en uno bidimensional

Lo que haremos será dividir un arreglo unidimensional en grupos según el valor del segundo parametro de la función:

    
    function dividirEnGrupos(array, division) {
    let newArr = [];
    for(let b = 0; b < array.length; b += division) {
    newArr.push(array.slice(b, b + division));
    }
    return newArr;
    }
    dividirEnGrupos([10, 20, 30, 40], 2);
    //Aquí nos devuelve: [[10, 20], [30, 40]]
