Programación Funcional

¿Qué es la programación funcional?

La programación funcional es un paradigma en la programación(también podemos decir que es un enfoque común del desarrollo de software) donde el código se organiza en funciones más básicas y pequeñas, las cuales se pueden combinar para construir programas más complejos
