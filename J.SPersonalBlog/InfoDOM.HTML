¿Qué es el DOM?

El Document Object Model(DOM) es la representación de los objetos(elementos) que conforman la estructura de un documento en la web. Es decir, representar el documento HTML mediante el DOM nos permite acceder y manipular sus elementos. En el DOM los elementos son representados como Nodos y Objetos, con los que podemos trabajar en JavaScript, esto incluye propiedades, métodos y eventos.

¿Qué es un Nodo?

Consiste en un punto específico del diagrama o del árbol de Nodos del DOM. Los elementos que incluyen los Nodos son:

  1. El Documento como tal.
  2. Los elementos HTML.
  3. El texto entre etiquetas.
  4. Los comentarios.

Jerarquía del DOM

  1. Root Node: Es el nodo úbicado en la parte superior del árbol. En HTML es <html>
  2. Parent Node: Es el nodo que contiene uno o más nodos hijos dentro de su estructura
  3. Child Node: Es cualquier nodo que está directamente contenido dentro de otro nodo (su nodo padre)
  4. Descendant Node: Es cualquier nodo que esté dentro de otro nodo, sin importar la profundidad
  5. Sibiling Node: Es cualquier nodo que comparte el mismo Parent Node que otro nodo

Diferencia entre Nodo vs Elemento

En el contexto del DOM, el concepto de nodo es más ámplio que el de elemento:

  • Usualmente nos referimos a los elementos del documento HTML como "elementos"
  • Un nodo puede ser un elemento HTML, pero también puede ser texto o comentarios en el documento

Seleccionar Elementos

.getElementById()

Método del objeto "documento" en J.S utilizado para seleccionar un elemento HTML especifíco mediante un atributo "id".Devuelve una referencia al primer elemento que coincida con el "id" dado o Null si no se encuentra ningún elemento con ese "id". Su sintaxis es:


    document.getElementById("id-del-elemento");

Ejemplo de uso:


    let elemento = document.getElementById("titulo");
    console.log(elemento.textcontent);

Contenido del elemento

  • innerHTML: módifica el contenido HTML interno.
  • textContent: módifica el texto del elemento, sin interpretar etiquetas HTML.
  • innerText: similar a "textContent", pero tiene en cuenta los estilos CSS.

    let titulo = document.getElementById("titulo");
    titulo.innerHTML = <b>Nuevo Titulo</b>
    titulo.textContent = "Nuevo Titulo";
