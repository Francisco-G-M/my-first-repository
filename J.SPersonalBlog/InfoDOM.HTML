Volver al principio

¿Qué es el DOM?

El Document Object Model(DOM) es la representación de los objetos(elementos) que conforman la estructura de un documento en la web. Es decir, representar el documento HTML mediante el DOM nos permite acceder y manipular sus elementos. En el DOM los elementos son representados como Nodos y Objetos, con los que podemos trabajar en JavaScript, esto incluye propiedades, métodos y eventos.

¿Qué es un Nodo?

Consiste en un punto específico del diagrama o del árbol de Nodos del DOM. Los elementos que incluyen los Nodos son:

  1. El Documento como tal.
  2. Los elementos HTML.
  3. El texto entre etiquetas.
  4. Los comentarios.

Ahora que ya sabes que es un nodo, te explícare brevemente la diferencia entre los Nodos con los elementos:

  • Usualmente nos referimos a los elementos del documento HTML como "elementos"
  • Un nodo puede ser un elemento HTML, pero también puede ser texto o comentarios en el documento

Jerarquía del DOM

  1. Root Node: Es el nodo úbicado en la parte superior del árbol. En HTML es <html>
  2. Parent Node: Es el nodo que contiene uno o más nodos hijos dentro de su estructura
  3. Child Node: Es cualquier nodo que está directamente contenido dentro de otro nodo (su nodo padre)
  4. Descendant Node: Es cualquier nodo que esté dentro de otro nodo, sin importar la profundidad
  5. Sibiling Node: Es cualquier nodo que comparte el mismo Parent Node que otro nodo

Seleccionar Elementos:

.getElementById()

Método del objeto "documento" en J.S utilizado para seleccionar un elemento HTML especifíco mediante un atributo "id".Devuelve una referencia al primer elemento que coincida con el "id" dado o Null si no se encuentra ningún elemento con ese "id". Su sintaxis es:


    document.getElementById("id-del-elemento");

Ejemplo de uso:


    let elemento = document.getElementById("titulo");
    console.log(elemento.textcontent);

También puedes combinarlo junto con otras etiquetas, agregando así nuevas funciones, por ejemplo:

Contenido del elemento

  • innerHTML: módifica el contenido HTML interno.
  • textContent: módifica el texto del elemento, sin interpretar etiquetas HTML.
  • innerText: similar a "textContent", pero tiene en cuenta los estilos CSS.

    let titulo = document.getElementById("titulo");
    titulo.innerHTML = <b>Nuevo Titulo</b>
    titulo.textContent = "Nuevo Titulo";

Estilos CSS

  • style: módifica estilos en línea como color, tamaño de fuente, etc.
  • className: cambia la clase CSS del elemento.
  • classList: añade, elimina o cambia clases dinámicamente.

    titulo.style.color = "red";
    titulo.style.fontSize = "24px";
    titulo.className = "Nueva-clase";
    titulo.classList.add("resaltado");

Atributos HTML

  • setAttribute(nombre, valor): módifica o agrega atributos.
  • getAttribute(nombre): obtiene valor de un atributo.
  • removeAttribute(nombre): elimina un atributo.

    let enlace = document.getElementById("enlace");
    enlace.setAttribute("href", "https:--------");
    console.log(enlace.getAttribute("href"));
    enlace.removeAttribute("target");

Eventos y Comportamientos

  • onclick: asigna una función al evento "click".
  • onmouseover: cambia el comportamiento cuando el mouse pasa sobre el elemento.
  • addEventListener(evento, función): forma más flexible de manejar eventos.

    titulo.onclick = funtion() {
        alert("Hiciste Click");
    };
    titulo.addEventListener("mouseover", function() {
        titulo.style.color = "blue";
    });

Mostrar u ocultar un elemento

  • style.display: cambia la visibilidad del elemento(none, block, inline, etc).
  • style.visibility: oculta el elemento sin quitarlo del flujo del documento.

    titulo.style.display = "none";
    titulo.style.visibility = "hidden";

.getElementsByClassName()

Método del objeto "document" o de cualquier elemento HTML que permite seleccionar todos los elementos que tiene una clase específica.Su sintaxis es:


    document.getElementByClassName("nombre-de-clase");
    elemento.getElementByClassName("--");

Características:

  • Devuelve una colección de elementos(HTMLCollection) que tienen la clase específicada.
  • Es una colección "en vivo", osea, si el DOM cambia(ya sea que se agregan/eliminan elementos con esa clase), la colección se actualizará automáticamente.
  • No devuelve un solo elemento, sino una colección, por lo que para acceder a un elemento especifíco, hay que usar índices, como si fuera un Array.
  • Si no hay elementos con esa clase, devuelve una colección vacía.

.getElementsByTagName()

Método del DOM que se usa para seleccionar elementos de una página web basándose en su etiqueta HTML.Retorna una colección "en vivo" de todos los elementos que coinciden con el nombre de la etiqueta específica.Su sintaxis es:


    document.getElementsByTagName("Tag-Name");
  • Donde esta "Tag-Name" va la cadena de texto que representa el nombre de la etiqueta que deseas seleccionar("p", "div", "span", etc).
  • Si bien devuelve una "HTMLCollection"(que es similar a un Array), no tiene todos los métodos de los Arrays

Dato Extra:

Si no hay coincidencias, devolverá una colección vacía(HTMLCollection con length = 0), pero no es Null.

.querrySelector()

Método que permite seleccionar el primer elemento del documento que coincida con un selector CSS específicado. Este sirve para acceder y manipular elementos del DOM de forma dinámica, lo que es útil para cambiar contenido, modificar estilos, agregar/eliminar clases, etc. Su sintaxis es:


    document.querrySelector("selectorCSS");
    document.querrySelector(".mi-clase");
    document.querrySelector("#mi-id");
    document.querrySelector("p");
    document.querrySelector(".lista li");

"selectorCSS": string que sigue la sintaxis de los selectores css.

  • "p": selecciona el primer <p>
  • ".lista li": primer <li> dentro de .lista

.querrySelectorAll()

Método que permite seleccionar múltiples elementos dentro de un documento HTML utilizando un selector CSS. Retorna una NodeList (lista de nodos parecida a un Array), conteniendo todos los elementos que coinciden con el selector privado. Sirve para:

  • Seleccionar todos los elementos que coinciden con el selector.
  • Aplicar modificaciones/estilos a múltiples elementos a la vez.
  • Recorrer los elementos seleccionados con métodos (e.j: forEch)

    document.querrySelectorAll("selector");

Principales propiedades/métodos para classList:

remove(class1, class2, ...)

Elimina una o más clases del elemento, si la clase no existe, no hace nada.

toggle(className, force)

Alterna la presencia de una clase. Si la clase existe, la elimina, si no existe, la añade. También acepta un segundo argumento booleano(force)

contains(className)

Verifica si el elemento tiene una clase específica, retorna "true" si la clase está presente, y "false" si no.

replace(oldClass, newClass)

Reemplaza una clase existente con una nueva. Si "oldClass" no existe, no hace nada.

forEach(callBack)

Este método solo esta disponible en navegadores modernos. Permite iterar sobre todas las clases del elemento.

length

Devuelve el número de clases en el elemento.

value

Devuelve todas las clases en formato de cadena separada por espacios.


    element.classList.remove("active", "highlight");
    element.classList.toggle("hidden");
    element.classList.toggle("hidden", true);
    element.classList.contains("active");
    element.classList.replace("old-class", "new-class");
    element.classList.forEach(cls => {
        console.log(cls);
    });
    console.log(element.classList.length);
    console.log(element.classList.value);

.createElement()

Función que crea nuevos elementos HTML dentro del DOM. Se invoca mediante el objeto "document" y permite generar dinámicamente elementos sin necesidad de escribir HTML directamente en el código. Sintaxis:


    let newElement = document.createElement("etiquetaHTML");

.appendChild()

Método utilizado para agregar un nodo como hijo de otro nodo en el DOM. Sintaxis:


    elementoPadre.appendChild(nuevoHijo);
