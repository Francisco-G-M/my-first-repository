¿Qué es la programación de algoritmos intermedios?

La P.A.I se centra en el desarrollo de soluciones avanzadas que van más allá de los conceptos básicos de la sintaxis y estructuras de control. La cual implica el uso de estructuras de datos más complejas, optimización de código, manipulación eficiente de datos y aplicación de paradigmas de programación.

Como sumar todos los numeros en un rango

En este caso, debemos devolver la suma del rango entre dos números de un array:


    function total(array) {
        let max = Math.max(array[0], array[1]);
        let min = Math.min(array[0], array[1]);
        let sumTotal = 0;
        for(let a = min; a <= max; a++) {
            sumTotal += a;
        }
        return sumTotal;
    }
    console.log(total([1, 99]));
    //Retorna: 4950

Math.max()

Método de "Math" que devuelve el mayor de los números proporcionados como argumentos. Sintaxis:


    Math.max(valor1, valor2, ...., valorN);

Si algún argumento no es un número, devuelve "NaN", pero si no se pasan argumentos, retorna "Infinity".

Math.min()

Método de "Math" que devuelve el número más pequeño de los proporcionados como argumentos:


    Math.min(valor1, valor2, ...., valorN);

Si algún argumento no es un número, devuelve "NaN", pero si no se pasan argumentos, retorna "Infinity".
