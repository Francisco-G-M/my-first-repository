¿Qué es la programación de algoritmos intermedios?

La P.A.I se centra en el desarrollo de soluciones avanzadas que van más allá de los conceptos básicos de la sintaxis y estructuras de control. La cual implica el uso de estructuras de datos más complejas, optimización de código, manipulación eficiente de datos y aplicación de paradigmas de programación.

Como sumar todos los numeros en un rango

En este caso, debemos devolver la suma del rango entre dos números de un array:


    function total(array) {
        let max = Math.max(array[0], array[1]);
        let min = Math.min(array[0], array[1]);
        let sumTotal = 0;
        for(let a = min; a <= max; a++) {
            sumTotal += a;
        }
        return sumTotal;
    }
    console.log(total([1, 99]));
    //Retorna: 4950

Math.max()

Método de "Math" que devuelve el mayor de los números proporcionados como argumentos. Sintaxis:


    Math.max(valor1, valor2, ...., valorN);

Si algún argumento no es un número, devuelve "NaN", pero si no se pasan argumentos, retorna "Infinity".

Math.min()

Método de "Math" que devuelve el número más pequeño de los proporcionados como argumentos:


    Math.min(valor1, valor2, ...., valorN);

Si algún argumento no es un número, devuelve "NaN", pero si no se pasan argumentos, retorna "Infinity".

Diferencia entre dos arrays

En esta ocasión se comparan dos arrays, y se devuelve uno nuevo con los elementos que sólo se encuentran en uno de los dos arrays brindados:


    function diferencia(arr1, arr2) {
        const newArr = [];
        function fusionar(primero, segundo) {
            for(let a = 0; a < primero.length; a++) {
                if(segundo.indexOf(primero[a]) === -1) {
                    newArr.push(primero[a]);
                }
            }
        }
        fusionar(arr1, arr2);
        fusionar(arr2, arr1);
        return newArr;
    }

La primera llamada "fusionar" busca los elementos de "arr1" que no están en "arr2" y lo agregamos a "newArr". La segunda llamada hace lo mismo pero con "arr2".

Método ".includes()"

Este método se utiliza para verificar si un array o una cadena de texto contiene un determinado valor. Su sintaxis es:


    array.includes(elemento, inicio);
    cadena.includes(elemento, inicio);

En caso de que el elemento si este devuleve "true", caso contrario será "false". El valor de "inicio" es opcional.

Busca y destruir elementos en un Array usando "filter" e "includes"

En este caso, debemos buscar en un array una "x" cantidad de argumentos de tipo "number", que posteriormente eliminaremos del array:


    function destruir(array, ...remover) {
        return array.filter(item => !remover.includes(item));
    }
  • .filter(): crea un nuevo array con los elementos que cumplen una condición.
  • .includes(): verifica si un valor está en el array "remover".
  • Negación "!": se usa para eliminar los elementos que están en "remover".
