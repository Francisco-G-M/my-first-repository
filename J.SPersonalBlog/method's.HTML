¿Qué son los Métodos en JavaScript?

En JavaScript, los métodos son una serie de instrucciones para completar una tarea/proceso, o una función asignada a un objeto.Estos mismos nos sirven mucho para ahorrar, simplificar y agilizar nuestro código.

push()

El método push() permite añadir un elemento al final de un Array:

    
    let estaciones = ['Invierno', "Otoño", 'Primavera'];
    estaciones.push("Verano");
    console.log(estaciones);

pop()

El método pop() sirve para remover el último elemento de un Array:

    
    let estaciones = ['Invierno', "Otoño", 'Primavera', "Verano"];
    estaciones.pop()
    console.log(estaciones);
    //También podemos asignarle el valor eliminado a una variable:
    var estacion;
    estacion = estaciones.pop();

shift()

Este método sirve para remover el primer elemento de un Array:

    
    let estaciones = ['Invierno', "Otoño", 'Primavera', "Verano"];
    estaciones.shift();
    console.log(estaciones);

unshift()

El método unshift() tiene como función agregar un elemento al principio de un Array:

    
    let estaciones = ["Otoño", 'Primavera', "Verano"];
    estaciones.unshift('Invierno');
    console.log(estaciones);

test()

Este método se utiliza para verificar la existencia de la expresión regular que le asignemos en una determinada string.En caso de que encuentre la expre.regular en la cadena devuelve true, de lo contrario devuelve false:

    
    let firstString = "Hi people";
    let otherString = /Hi/;
    //La expresión regular es /Hi/, esta no va con ningún tipo de comillas dentro
    let result = otherString.test(firstString);
    //Oh también:
    otherString.test(firstString);

match()

Con el método match() podemos extraer coincidencias reales:

    
    let extraer = "Extraer la palabra codigo";
    let extracto = /codigo/;
    const resultado = extraer.match(extracto);
    //Dato extra: tanto aquí como en ejemplos anteriores le estamos asignando el valor de la expre.regular "codigo" a la variable "resultado", esto es 
    opcional como ya hemos visto.

Tipos de métodos con console

Fuera de "console.log()" existen otros tipos de métodos para console,e.j:

console.warn()

Su función es imprimir texto en la consola como una advertencia.

console.error()

Su función es mostrar un mensaje de error en la consola.

console.info()

Muestra en la consola un mensaje informativo.

console.log()

El console.log() imprime texto en la consola como un mensaje de registro.

    
    console.warn('Esto es una advertencia');
    console.error('Esto es un error');
    console.info('Esto es un mensaje informativo');
    console.log('Esto es un mensaje de registro o un punto de control del código');

splice()

En el caso de que quisieramos eliminar cualquier número de elementos consecutivos de cualquier parte de matriz.Este puede tomar tres parametros:

  1. El primero representa el índice de la matriz desde la cual comenzar a eliminar elementos
  2. El segundo índica el número de elementos a eliminar
  3. El tercer parametro nos permite agregar elementos a la matriz
    
    //Ejemplo de los primeros dos parametros:
    let string = ['que', 'onda', 'capo', 'todo', 'bien'];
    string.splice(2, 2);
    //Aquí estamos eliminando 2 elementos comenzando con el tercer elemento(en el índice 2).Ahora string tiene de valor ['que', 'onda', 'bien']
    //También podemos asignarle los valores eliminados a una variable:
    let string2 = string.splice(2, 2);
    //Ahora string2 es igual a : ['pa', 'todo'], también podemos usar varios splice() :
    const numbers = [1, 2, 3, 4, 5, 6, 7, 8];
    numbers.splice(0, 2);
    numbers.splice(1, 2);
    numbers.splice(2, 2);
    console.log(numbers);
    //Aquí eliminamos: 1,2,4,5,7 y 8, solo quedando [3, 6]
    /*----------------------------------------------------------------------------------------------------------------------------------------------*/
    //Ejemplo del tercer parametro:
    const numbersArray = [10, 20, 30, 30, 60];
    const index = 3;
    const remove = 1;
    numbersArray.splice(index, remove, 40, 50);
    console.log(numbersArray);
    //Esta llamada nos devolvería: [10, 20, 30, 40, 50, 60] siendo este Array el nuevo valor de numbersArray.

Object.keys()

Su función es generar un Array que tenga todas las claves almacenadas en un objeto.Dentro de los parentesis de este método escribimos el nombre del objeto como argumento:

    
    let family = {
    John: {
        character: "Son"
    },
    Mike: {
        character: "Father"
    },
    Mariana: {
        character: "Mather"
    }
    };
    let result = Object.keys(family);
    console.log(result);
    function myFamily(member) {
    return Object.keys(member);
    };
    //Tanto la llamada de la función y el de result devuelven : ['John', 'Mike', 'Mariana']

slice()

Su función es copiar/extraer una cantidad determinada de elementos a una nueva matriz, sin provocar una mutación el la matriz original:

    
    let originalArray = ['one', 'two', 'three', 'four', 'five'];
    let newArray = originalArray.slice(1, 3);
    //Aquí originalArray mantiene su Array y ahora newArray tiene de valor ['two', 'three']
    //ACLARACIÓN: la extracción no incluira el elemento perteneciente al último índice de slice(), por eso no figura en newArray 'four'

indexOf()

Este método sirve para comprobar rápida y facilmente la presencia de un elementoen una matriz, este toma un elemento de la matriz como parámetro y cuando se llama devuelve la posición/índice de ese elemento, si el elemento no existe en la matriz devuelve -1 :

    
    let alimentos = ['Frutas', 'Pastas', 'Verduras', 'Lacteos'];
    console.log(alimentos.indexOf('bebidas'));
    console.log(alimentos.indexOf('Frutas'));
    console.log(alimentos.indexOf('Verduras'));
    console.log(alimentos.indexOf('Lacteos'));
    //La primera llamada devuelve -1, la ssegunda 0, la tercera 2 y la cuarta 3

filter()

El método filter() tiene como función crear un nuevo Array con los elementos que cumplan con la condición dada (.filter(condición)), en este caso, la condición son los valores Booleanos primitivos(true, false), devolviendonos solo los elementos que sean true:

    
    function convertirValoresEnBooleanos(array) {
    return array.filter(boolean);
    };
    let a = convertirValoresEnBooleanos([10, "Hi", NaN, 9, false]);
    console.log(a);
    //Aquí nos devuelve los elementos considerados true: [10, 'Hi', 9] , todos los demas son false
