¿Qué es JavaScript?

Es un lenguaje de programación utilizado para volver más interactiva a una página web. En el encontraras herramientas como: funciones,variables,objetos,arreglos,sentencias,ciclos,etc.

Comentarios

Son parte del código que se ignora(que no figura en la consola),sirve para comentar el código. Básicamente cumple una función tipo notas que dejamos een nuestro código para explicar alguna parte de nuestro código,marcar alguna expresión,agregar datos extra,etc. Existen dos tipos de comentarios:

  1. Comentario de una línea:
  2. //esto es un comentario
  3. Comentario de varias líneas:
  4. /*esto es
  5. un comentario*/
