BasicJavaScript ES6

¿Qué es JavaScript?

Es un lenguaje de programación utilizado para volver más interactiva a una página web. En el encontraras herramientas como: funciones,variables,objetos,arreglos,sentencias,ciclos,etc.

Comentarios

Son parte del código que se ignora(que no figura en la consola),sirve para comentar el código. Básicamente cumple una función tipo notas que dejamos een nuestro código para explicar alguna parte de nuestro código,marcar alguna expresión,agregar datos extra,etc. Existen dos tipos de comentarios,e.j:

  1. Comentario de una línea:
  2. //esto es un comentario
  3. Comentario de varias líneas:
  4. /*esto es
  5. un comentario*/

Variables

Las variables crean un espacio en la memoria del dispositivo para que guardemos un valor específico en ese lugar de memoria,e.j:

  1. var miVariable = "hi";
  2. console.log(miVariable);

También podes aisnarle un valor númerico,e.j:

  1. miVariable = 16;

Asignar el valor de una var a otra var

Para poder asignarle el valor de una var a otra var hariamos así :

  1. var a = 5;
  2. var b = a;

y lo confirmamos con : console.log(a) , y lo mismo con b.Otra forma más sencilla sería : b = a; , esto resulta más práctico.

Inicializar variables

Básicamente, cuando le asignas un valor a la var,la estas inicializando,e.j: var x = 9; miIdioma = "Español";

Variables no inicializadas

Se refiere a una variable que no tiene un valor asignado,e.j:

var a;

Mayúsculas y miúsculas

Al ser JavaScript un lenguaje "sencible" con las mayúsculas y minúsculas, no podemos escribir diferente el nombre de la variable al momento de llamarla,e.j:

var miVariable = 5;

console.log(MIVARIABLE)

Suma,resta,multiplicación y división

Para sumar en J.S es muy simple,e.j: var suma = 7 + 12;

Para la resta,multiplicación y división es básicamente lo mismo,e.j: var resta = 15 - 5; var producto = 5*3; var cociente = 20/2;

Un dato a destacar es que si divides entre Cero, te figurara Infinity en la consola.

Números decimales en J.S

En J.S al escribir números decimales usamos un punto para separar la parte decimal de la entera, e.j: var cuentaTotal = 23.4;

multiplicar y dividir números decimales

Es lo mismo que cuando lo hacemos con números enteros.e.j: var producto = 3.4*10.4; var cociente = 5.0/2.0;

Resto de una división

Para calcular el resto de una división usamos el signo " % " ,e.j: var resto = 15 % 5;

Incrementar el valor de una variable

Para incrementar el valor de una variable hay dos maneras :

Opción 1:

var incrementar = 1; incrementar + 1;

Opción 2:

Esta opción solo funciona para incrementar el valor en 1: incrementar++;

Reducir el valor de una variable

Para reducir el valor de una variable hay dos maneras:

Opción 1:

var numeroDeEstudiantes = 100; numeroDeEstudiantes = numeroDeEstudiantes - 1;

Opción 2:

Otra forma más práctica sería : numeroDeEstudiantes --; pero esta opción solo sirve para reducir de a uno.

Asignación de suma, resta, multiplicación y división

Sirve para incrementar los valores en más de uno,e.j: var a = 10; a = a + 5; a = a - 5; a = a * 5; a = a / 5;

Otra forma más práctica sería

a += 5; a -= 5; a *= 5; a /= 5;

Variables con cadenas de caracteres

Estas se usan para representar textos en el código,e.j: var nombre = "Alan";

Las cadenas de caracteres se rodean con comillas dobles o simples.Lo que si no podemos hacer es empezar la cadena con un tipo de comillas y terminarla con otro tipo.

Escapar comillas en cadenas de caracteres

Sirve para "escapar" comillas en cadenas, básicamente omitirla,e.j: var myString = "Escapar\"comillas dobles"";

En este caso la barra " \ " sirve para escapar las comillas dobles , el resultado: Escapar "comillas dobles".

Cadenas de caracteres con comillas simples

Podemos usar comillas dobles dentro de comillas simple y viceversa,e.j: var miMeta; miMeta = 'Aprender a programar con "JavaScript"';

Si usas '' por afuera podes usar "" por dentro y viceversa.

Secuencias de escape

Son una combinación de de caracteres que no se representan a sí mismos en una cadena de caracteres:

Tipos de secuencias de escape

Comilla simple( \' ), Comilla doble( \" ), Barra invertida( \\ ), Nueva línea( \n ), Retorno de carro( \r ), Tabulación( \t ), Retroceso( \b ) y Salto de página( \f ).e.j:

console.log("Estoy aprendiendo\"JavaScript""); o también: console.log("Estoy aprendiendo:\n JavaScript");

Concatenar cadenas de caracteres

Para poder unir cadenas de caracteres debemos usar el signo + ,e.j: var nombreCompleto = "Alan" + " " + "Turing";

Las comillas vacias son para que entre el nombre y el apellido tenga un espacio que los distinga.

Otra forma sería

var nombreCompleto = "Alan" + "Turing"; Aquí los dos nombres figuraran juntos.

Construir cadenas de caracteres con variables

También se pueden construir cadenas de caracteres con variables,e.j: var verbo = "programar"; var mensaje = "Estoy aprendiendo a " + verbo; console.log(mensaje);

También puedes dejar un espacio al final de la cadena a la que le vas a concatenar algo, resulta siendo más práctico.Puedes concatenar todas las cadenas que quieras.

Agregar variables a cadenas de caracteres

El procedimiento de agregar variables a cadenas de caracteres es muy simple, mira el siguiente ejemplo:

var mensajeCompleto = "Estoy aprendiendo a programar"; var parteFinal = "con JavaScript"; mensajeCompleto += parteFinal;

Básicamente lo que hacemos es escribir el nombre de la variable a la que le queremos concatenar otra cadena más el nombre de la cadena a concatenar, entre ellos el signo +=
