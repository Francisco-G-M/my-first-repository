¿Qué es la programación de algoritmos básicos?

La programación de algoritmos básicos se basan en los fundamentos del pensamiento algorítmico,.A su vez estos algoritmos básicos son un conjunto ordenado y determinado de operaciones que permiten la solución de un problema.

convertir Celsius a Fahrenheit

Para convertir Celsius a Fahrenheit usamos la siguiente formula: Celsius múltiplicado por 9/5 + 32:

    
    function convertirTemperaturas(celsius) {
    const FAHRENHEIT = celsius*9/5 + 32;
    return FAHRENHEIT;
    }
    let final = convertirTemperaturas(30);
    console.log(final);

Invertir una cadena

En caso de que quisieramos invertir una cadena podemos hacerlo mediante un ciclo for:

    
    function invertir(string) {
    let newArr = "";
    for(let i = string.length - 1; i >= 0; i--) {
    newArr += string[i]; 
    }
    return newArr;
    }
    let final = invertir("hola");
    console.log(final);
